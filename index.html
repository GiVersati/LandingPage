UI designer, ou designer de interface do usuário

O meu objetivo como UI design é tornar a interação digital o mais simples, fluida, intuitiva e eficiente possível.

A interface de usuário é um conjunto de controles e canais sensoriais pelos quais um usuário pode comunicar-se com uma máquina.

O UI Designer cuida daquela parte em que o usuário interage com um site (no caso do web design), layout ou produto.
